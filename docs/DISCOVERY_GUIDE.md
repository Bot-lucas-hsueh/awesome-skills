### 「我应该用哪个技能？」快速判断流程

```
START
  ↓
选择你的角色/职位
  ├─→ 高管/CEO → [Executive Skills]
  ├─→ 工程师 → [Software Skills]
  ├─→ 医生/医疗 → [Healthcare Skills]
  ├─→ 财务/投资 → [Finance Skills]
  ├─→ 销售/市场 → [Sales & Marketing]
  └─→ 其他 → [Browse by Domain]
  ↓
选择你的具体任务
  ├─→ 制定战略/决策 → CEO + Strategy Skills
  ├─→ 代码开发/审查 → Engineer + Code Review
  ├─→ 数据分析 → Data Scientist + Analytics
  ├─→ 内容创意 → Writer + Creative
  └─→ 客户管理 → Sales + Account Manager
  ↓
查看技能详情
  ├─→ SKILL.md（主要说明）
  ├─→ references/（深度内容）
  └─→ EVALUATION_REPORT.md（质量评分）
  ↓
END (安装技能)
```

### 新手入门路线图

**第1周：基础**
- Day 1-2: 安装 `CEO` 技能（学习决策框架）
- Day 3-4: 安装 `Product Manager` 技能（学习用户思维）
- Day 5-7: 安装 `Engineer` 技能（学习技术思维）

**第2周：深化**
- 安装行业相关技能（金融/医疗/销售等）
- 尝试组合使用（例：CEO + Engineer = 创始人）

**第3周：高级**
- 用 Skill Writer 创建自己的技能
- 用 MCPs 连接外部工具
- 建立个人技能组合

### 常见问题

**Q: 技能和 Prompt 的区别？**
A: 技能 = Prompt + 参考资料 + 工作流 + 质量保证

**Q: 能同时用多个技能吗？**
A: 能！用 Graph of Skills 组合多个技能成工作流

**Q: 我的平台不支持怎么办？**
A: 查看 [支持平台列表](./INSTALL-GUIDE.md#支持的平台)

**Q: 技能更新了怎么办？**
A: 自动同步，无需手动更新
