# 航空航天与交通职业路径 Aerospace & Transportation Roadmap

> 覆盖方向：航空工程师 · 飞行员 · 自动驾驶工程师 · 物流经理 · 交通规划

---

## 职业全景

```
航空航天与交通
├── 航空航天
│   ├── 航空工程师 → 高级工程师 → 主任工程师 → 总工程师
│   ├── 飞行员 → 副机长 → 机长 → 教员 → 航务总监
│   └── 航天工程师 → 系统集成工程师 → 总体设计师
├── 汽车与智能交通
│   ├── 汽车工程师 → 整车架构师 → 技术总监 → CTO
│   ├── 自动驾驶工程师 → 系统架构师 → 自动驾驶总监
│   └── 新能源汽车: 电池/电机/电控工程师 → 三电总工
├── 交通规划
│   ├── 交通规划师 → 高级规划师 → 交通总工 → 城市交通局长
│   └── 轨道交通工程师 → 系统专家 → 工程总监
└── 物流运营
    ├── 物流专员 → 物流经理 → 物流总监 → COO
    └── 供应链分析师 → 供应链经理 → CSCO
```

---

## 🌱 第1级 · 入门

### 航空工程入门

**基础学科**
- [ ] 空气动力学：升力/阻力/伯努利原理
- [ ] 结构力学：应力应变、疲劳分析
- [ ] 推进系统：涡扇/涡桨/火箭发动机原理
- [ ] 飞行力学：稳定性和控制基础
- [ ] 适航标准：CCAR/FAR适航规章基础

**工具技能**
- [ ] CAD/CAM：CATIA / SolidWorks
- [ ] CFD仿真：ANSYS Fluent基础
- [ ] 结构分析：NASTRAN / Abaqus入门
- [ ] 复合材料基础：碳纤维/蜂窝结构

### 飞行员入门路径

```
路径一（航空公司）：
  PPL（私照）→ ATPL理论课程 → CPL（商照）→ IR（仪表）→ ME（多发）→ 航司招募（MPL/CADET）

路径二（军转民）：
  军队飞行员 → 退役转商业航空 → 民航改装训练

关键证照（中国民航）：
  PPL → CPL → ATPL（航线运输驾驶员执照）
  仪表等级（IR）+ 多发等级（ME）必须
```

### 自动驾驶工程师入门
- [ ] ROS（机器人操作系统）基础
- [ ] 传感器融合基础：相机/LiDAR/Radar
- [ ] 路径规划算法：A*、Dijkstra、RRT
- [ ] Python/C++编程（后者更重要）
- [ ] 机器学习在感知中的应用

### 关键里程碑
- ✅ 完成航空/汽车公司实习，接触真实工程项目
- ✅ 飞行员：取得PPL（私人飞行员执照）
- ✅ 自动驾驶：能运行一个基础的自动驾驶仿真Demo

## 🌿 第2级 · 初级 (2-5年)

### 航空结构工程师
- [ ] 复合材料详细设计：铺层顺序、工艺选择
- [ ] 疲劳/损伤容限（F&DT）分析
- [ ] 制造工艺：RTM/热压罐/自动铺带（ATL/AFP）
- [ ] 试验验证：地面强度试验、静强度试验
- [ ] DO-160 / CS-25 适航符合性验证

### 飞行员进阶
- [ ] 取得CPL + IR + ME（多发仪表商照）
- [ ] 航空公司MPL培训或外聘招募
- [ ] 副机长资格：机型改装→路线检查→放飞
- [ ] CRM（机组资源管理）：团队协作和决策
- [ ] UPRT（非正常姿态恢复）训练

### 自动驾驶进阶

**感知方向**
- [ ] 目标检测：YOLO、Transformer-based检测
- [ ] 3D点云处理：PointNet、PointPillars
- [ ] 多传感器融合：卡尔曼滤波、深度融合

**规划与控制方向**
- [ ] 行为预测：轨迹预测模型
- [ ] 运动规划：Frenet坐标系规划
- [ ] 控制算法：MPC（模型预测控制）
- [ ] 高精地图：HD Map的生产和消费

**仿真验证**
- [ ] CARLA / SUMO仿真平台
- [ ] 场景库构建：corner case挖掘
- [ ] 安全性验证方法：形式化验证、统计测试

### 物流/供应链初级
- [ ] 仓储管理：WMS系统（库位规划、收发货流程）
- [ ] 运输管理：TMS系统，承运商管理
- [ ] 最后一公里：城市配送、路径优化算法
- [ ] 3PL/4PL管理：外包物流管理

### 关键里程碑
- ✅ 飞行员：取得航线运输机型签注，独立执飞航线
- ✅ 自动驾驶：主导一个模块（感知/规划/控制）的开发
- ✅ 物流：独立管理一个仓库或运输网络节点

---

## 🌳 第3级 · 中级 (5-10年)

### 航空系统工程师

**系统集成**
- [ ] ARP 4754A：民机系统开发标准
- [ ] DO-178C / DO-254：机载软件/硬件适航
- [ ] 系统安全分析：FHA/FMEA/FTA
- [ ] 电传操控（FBW）系统理解
- [ ] 航电系统集成：IMA（综合模块化航电）

**项目管理**
- [ ] EVMS（挣值管理）：进度/成本管控
- [ ] 适航取证项目：与CAAC沟通和TC取证
- [ ] 供应商管理：AS9100质量体系

### 机长级飞行员

- [ ] 机长资格取得：机长检查→监控机长→正式机长
- [ ] TRI/TRE培训资质（有意向培训别人）
- [ ] 航务管理：参与航司安全委员会

### 自动驾驶总监预备

- [ ] 全栈自动驾驶系统设计：感知→预测→规划→控制→执行
- [ ] L4级功能开发主导（特定场景）
- [ ] 法规理解：GB/T 34590、ISO 26262（功能安全）
- [ ] 量产经验：从实验室到量产车的整个过程

### 关键里程碑
- ✅ 主持一个适航取证项目或子系统取证
- ✅ 自动驾驶：成功将L4功能在特定场景量产落地
- ✅ 带领10+人团队

---

## 🦅 第4级 · 高级 (10-20年)

### 总设计师 / 型号负责人

- [ ] 主导飞机/航天器/汽车新型号立项
- [ ] 技术方案决策：系统架构定义
- [ ] 跨专业协调：气动/结构/推进/航电/适航
- [ ] 与适航当局高层沟通
- [ ] 项目时间/成本/质量的最终责任人

### 自动驾驶CTO

- [ ] L4/L5自动驾驶商业化路线制定
- [ ] 技术团队管理（100+工程师）
- [ ] 融资和战略合作（Tier 1/OEM合作）
- [ ] 监管参与：自动驾驶法规制定建议

---

## 👑 第5级 · 精英

### 航空航天精英
- 🏆 **型号总师**：主导百亿级大型飞机/航天器研制
- 🏆 **中国工程院院士**：航空/航天领域最高荣誉
- 🏆 **宇航员**：进入太空执行任务（中国、NASA、商业）
- 🏆 **商业航天创始人**：建立民营火箭公司

### 自动驾驶精英
- 🏆 **CTO/CEO**：主导Robotaxi或自动驾驶独角兽
- 🏆 **技术突破**：发明解决关键技术难题的算法
- 🏆 **行业标准制定者**：主导国际自动驾驶安全标准
