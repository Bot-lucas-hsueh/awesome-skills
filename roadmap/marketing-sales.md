# 市场营销与销售职业路径 Marketing & Sales Roadmap

> 覆盖方向：市场经理 · 数字营销 · 增长黑客 · 品牌 · 销售经理 · 商务拓展

---

## 职业全景

```
市场营销与销售
├── 市场营销
│   ├── 市场专员 → 市场经理 → 市场总监 → CMO
│   ├── 品牌专员 → 品牌经理 → 品牌总监
│   └── 数字营销专员 → 增长经理 → 增长总监 → CGO
├── 销售
│   ├── 销售代表 → 区域销售经理 → 大区总监 → 销售VP
│   ├── 企业销售(B2B): SDR → AE → 企业销售总监
│   └── 零售销售: 店员 → 店长 → 区域经理 → 运营总监
├── 商务拓展
│   ├── BD专员 → BD经理 → 商务总监 → VP BD
│   └── 战略合作经理 → 战略合作总监
└── 内容营销
    ├── 内容编辑 → 内容总监 → 内容战略官
    └── SEO专员 → 增长黑客
```

---

## 营销与销售的核心差异

| 维度 | 市场营销 | 销售 |
|------|---------|-----|
| 时间视角 | 长期品牌/需求创造 | 短期成交/配额达成 |
| 衡量指标 | MQL、品牌认知、ROI | 收入、胜率、配额达成率 |
| 工作对象 | 批量受众 | 个体客户 |
| 核心技能 | 创意、数据分析、策略 | 关系建立、谈判、韧性 |

---

## 🌱 第1级 · 入门 (0-12个月)

### 市场营销入门
- [ ] 营销基础：4P理论（产品/价格/渠道/推广）、STP模型
- [ ] 数字营销基础：SEO/SEM/社交媒体/邮件营销
- [ ] 内容创作：文案写作、基础设计（Canva）、视频脚本
- [ ] 数据分析：Google Analytics / 百度统计基础
- [ ] CRM工具：Salesforce / HubSpot 基础使用
- [ ] 广告平台：Google Ads / Meta Ads / 抖音投放基础

### 销售入门
- [ ] 产品知识：彻底理解你卖的产品/服务
- [ ] 销售基础流程：开发→资格认定→演示→谈判→成交→客户服务
- [ ] 电话/邮件开发技巧（Cold Call / Cold Email）
- [ ] CRM工具使用（记录客户信息、跟进记录）
- [ ] 拒绝处理：常见异议和应对话术

### 关键里程碑
- ✅ 独立策划并执行一个小型营销活动（哪怕是社交媒体活动）
- ✅ 完成第一笔销售成交
- ✅ 建立并维护个人的客户/联系人数据库

### 常见陷阱 ⚠️
- 营销：只关注创意，忽视数据和ROI
- 销售：过度依赖话术，不真正理解客户需求
- 两者共同：不懂产品就很难有说服力

## 🌿 第2级 · 初级 (1-3年)

### 数字营销专员进阶

**付费流量（SEM/社交广告）**
- [ ] Google Ads：关键词研究、出价策略、质量得分优化
- [ ] 程序化广告：DSP平台、人群定向、再营销
- [ ] 社交广告：Facebook/抖音广告投放与优化
- [ ] A/B测试：广告创意、落地页、受众测试

**SEO / 内容营销**
- [ ] 技术SEO：网站速度、结构化数据、Core Web Vitals
- [ ] 关键词策略：搜索意图分析、内容规划
- [ ] 链接建设（Link Building）基础策略

**数据分析**
- [ ] 归因模型：理解多触点归因（First/Last Click / Data-Driven）
- [ ] UTM参数：精确追踪各渠道效果
- [ ] Dashboard搭建：Looker Studio / Tableau

### 增长黑客（Growth Hacker）方向
- [ ] AARRR框架：获取→激活→留存→变现→推荐
- [ ] 增长实验设计：假设→实验→测量→学习
- [ ] 产品分析：Mixpanel / Amplitude事件追踪
- [ ] 用户增长策略：病毒传播、推荐计划、钩子设计

### B2B销售进阶
- [ ] MEDDIC框架：Metrics/Economic Buyer/Decision/Competition/Identify Pain/Champion
- [ ] 价值销售：从功能销售转向ROI/业务价值销售
- [ ] 演示技巧：定制Demo，解决特定痛点
- [ ] 谈判技巧：让步策略、不让步的理由

### 关键里程碑
- ✅ 独立管理一个渠道（如SEO或SEM）并改善ROI
- ✅ 完成年度销售配额的120%
- ✅ 建立可重复的销售流程（Playbook）

---

## 🌳 第3级 · 中级 (3-6年)

### 市场总监能力

**战略营销**
- [ ] 去市场策略（GTM）：产品上市策略制定
- [ ] 竞争定位：竞争格局分析、差异化定位
- [ ] 品牌架构：主品牌/子品牌体系设计
- [ ] 营销漏斗全链路优化

**需求生成（Demand Generation）**
- [ ] ABM（基于账户的营销）：针对高价值目标账户
- [ ] 销售与营销对齐（Sales & Marketing Alignment）
- [ ] 营销预算分配优化
- [ ] MQL → SQL → 商机转化率提升

**品牌建设**
- [ ] 品牌策略：品牌价值主张、品牌个性
- [ ] 创意项目管理：Agency协作、Campaign执行
- [ ] 公关与媒体关系：新闻稿、媒体pitch
- [ ] 用户社区建设

### 销售经理/总监能力
- [ ] 销售团队建设：招聘、培训、激励机制
- [ ] 销售预测（Forecasting）：Pipeline管理
- [ ] 销售Playbook制定：标准化最佳实践
- [ ] 大客户管理（Key Account Management）
- [ ] 合作伙伴渠道建设（Channel Sales）

### 关键里程碑
- ✅ 主导一个完整的产品上市（GTM）项目
- ✅ 带领10人以下团队超额完成指标
- ✅ 建立数据驱动的营销决策体系

---

## 🦅 第4级 · 高级 VP / 总监 (6-12年)

### VP Marketing
- [ ] 公司级营销战略（全渠道、全生命周期）
- [ ] 营销科技栈（MarTech Stack）规划和治理
- [ ] 用户增长策略（PLG/SLG/MLG混合模式）
- [ ] 跨国市场营销经验（本土化策略）
- [ ] 年度营销预算1000万+管理

### VP Sales
- [ ] 销售组织设计（团队结构、区域划分）
- [ ] 年度配额制定和分解
- [ ] 渠道策略：直销/经销商/合伙人体系
- [ ] 关键客户亲自维护（战略客户）
- [ ] 收入运营（RevOps）体系建立

### 关键里程碑
- ✅ 管理50人以上营销/销售组织
- ✅ 带领公司收入规模突破重要里程碑
- ✅ 推动品牌认知度显著提升（可量化）

---

## 👑 第5级 · CMO / 销售精英

### CMO核心成就
- 🏆 主导品牌从区域到全国/全球的突破
- 🏆 PLG策略成功推动产品增长（用户基础10x+）
- 🏆 数字营销ROI超越行业平均2倍以上
- 🏆 打造行业标杆营销案例（获行业奖项）

### 顶级销售领袖
- 🏆 带领销售团队完成10亿+年收入
- 🏆 建立可复制的销售组织（快速扩张）
- 🏆 将新市场/新产品从0做到行业第一名
