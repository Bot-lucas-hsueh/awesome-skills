# 农业与食品职业路径 Agriculture & Food Roadmap

> 覆盖方向：农业技术员 · 农场管理 · 食品科学 · 农业顾问 · 农业科技

---

## 职业全景

```
农业与食品
├── 农业技术
│   ├── 农业技术员 → 高级技术员 → 农业推广专家 → 首席农业顾问
│   ├── 植保专员 → 植保工程师 → 农药/农资专家
│   └── 土壤科学家 → 高级土壤顾问 → 土壤研究员
├── 农场管理
│   ├── 农场工人 → 班组长 → 农场场长 → 区域农业总监
│   ├── 家庭农场主 → 合作社理事长 → 农业企业总经理
│   └── 畜牧技术员 → 兽医 → 牧场总监
├── 食品科学与加工
│   ├── 食品检验员 → 食品工程师 → 食品研发总监
│   ├── 食品安全专员 → 质量总监 → 食品安全总监
│   └── 营养师 → 临床营养师 → 营养顾问
├── 农业科技（AgTech）
│   ├── 精准农业技术员 → AgTech解决方案经理 → AgTech总监
│   ├── 农业无人机操作员 → 农业航测工程师
│   └── 垂直农业工程师 → 植物工厂技术总监
└── 农业政策与研究
    └── 农业推广员 → 农业政策分析师 → 农业发展规划专家
```

## 🌱 第1级 · 入门 (0-2年)

### 农业技术员入门

**基础农学知识**
- [ ] 作物学基础：主要粮食/经济作物的生长规律
- [ ] 土壤科学：土壤类型、pH值、养分循环
- [ ] 植物保护：病害/虫害/草害的识别与防治
- [ ] 肥料基础：大量元素/中微量元素的作用与施用
- [ ] 灌溉技术：漫灌/滴灌/喷灌的适用场景

**实操技能**
- [ ] 农业机械操作：拖拉机、播种机、收割机基础
- [ ] 农药安全使用：配比、防护、安全间隔期
- [ ] 农情记录：田间档案建立、生育期记录
- [ ] 气象判断：基本天气识别，利用天气预报指导农事

### 食品安全/检验入门
- [ ] 食品法规：GB 2762/2761（污染物/农药残留限量）
- [ ] 检测技术：HPLC/GC基础、微生物检测
- [ ] HACCP体系：关键控制点识别
- [ ] 食品标签：GB 7718食品标签通则

### 关键里程碑
- ✅ 完整跟踪一个完整作物生产季（播种→收获）
- ✅ 独立制定一份病虫害防治方案
- ✅ 参与一次农产品质量检测

### 常见陷阱 ⚠️
- 忽视实地观察：农业需要在田间积累经验，书本知识不够
- 过于依赖化学投入品：可持续农业趋势下，需了解绿色方案
- 不懂农业经济：好技术也要算经济账

## 🌿 第2级 · 初级 (2-5年)

### 农业技术推广员

**技术能力**
- [ ] 测土配方施肥：土壤化验→分析→配肥方案
- [ ] 绿色防控：生物防治/物理防治/化学防治综合运用
- [ ] 新品种评价：田间试验设计、数据分析（Excel/SPSS）
- [ ] 水肥一体化：设备安装调试、灌溉计划制定
- [ ] 农业保险：政策性农业保险理解和应用

**推广技能**
- [ ] 农民培训：能将技术简单化，农民听得懂
- [ ] 示范田建设：对比展示新技术效果
- [ ] 农技服务：大田巡回指导，解答农民问题
- [ ] 农情监测：苗情/墒情/病虫情的系统监测

### 农场管理员

**生产管理**
- [ ] 年度生产计划：种植计划→物资计划→劳动力计划
- [ ] 成本核算：每亩/每吨的生产成本精确计算
- [ ] 农机调度：多台机械高效协调作业
- [ ] 农产品品控：收获品质管控、分级包装

**农业经营**
- [ ] 销售渠道：批发市场/超市直供/电商农业
- [ ] 品牌农业基础：有机认证、GAP认证申请
- [ ] 政策补贴：申请各类农业支持政策

### AgTech（农业科技）初级

- [ ] 无人机应用：DJI农业无人机飞防作业
- [ ] 遥感基础：卫星/无人机影像解译（NDVI指数）
- [ ] 精准农业：变量施肥/变量灌溉的概念与设备
- [ ] 农业物联网：传感器数据采集和平台对接

### 关键里程碑
- ✅ 负责500亩以上农田的技术指导
- ✅ 主持完成一项农业技术推广项目
- ✅ 获得农艺师/助理农艺师职称

---

## 🌳 第3级 · 中级 (5-10年)

### 高级农业技术专家

**专业深化（选择方向）**

**作物育种方向**
- [ ] 分子育种：分子标记辅助选择（MAS）
- [ ] 品种登记：新品种区域试验、品种审定流程
- [ ] 种质资源：遗传资源收集和保存

**植保/农药方向**
- [ ] 农药登记：登记资料准备、毒理实验理解
- [ ] 抗药性管理：轮换用药策略
- [ ] 生物农药：微生物/植物源农药研发趋势

**土壤健康方向**
- [ ] 有机质提升：秸秆还田、有机肥施用策略
- [ ] 盐碱地改造：物理/化学/生物改良技术
- [ ] 碳汇农业：土壤碳固定的测量和认证

### 农业企业管理

**经营管理**
- [ ] 供应链管理：从农场到餐桌的全链条
- [ ] 合同农业：订单农业合同设计
- [ ] 农业金融：农业贷款、农业保险、融资租赁
- [ ] 出口农业：出口认证、植物检疫、包装标准

**品牌建设**
- [ ] 地理标志产品：申请与运营
- [ ] 有机认证：GB/T 19630、USDA Organic
- [ ] 农业电商：拼多多/抖音农业直播带货

### 精准农业专家

- [ ] 农田信息系统：ArcGIS/QGIS农业应用
- [ ] 作物模型：DSSAT/APSIM作物生长模拟
- [ ] 机器学习应用：病虫害AI识别、产量预测
- [ ] 垂直农业：LED照明/水培/气雾栽培设计

### 关键里程碑
- ✅ 推广面积达到1万亩以上（推广专家方向）
- ✅ 管理年产值1000万+的农业基地
- ✅ 获得高级农艺师职称

---

## 🦅 第4级 · 高级 (10-20年)

### 首席农业顾问 / 农业科学家

**研究引领**
- [ ] 主持国家/省级农业科研项目
- [ ] 在核心期刊发表农业科研论文
- [ ] 参与国家农业标准、技术规程制定
- [ ] 新品种/新技术发明专利申请

**行业影响**
- [ ] 在农业协会担任理事/专家委员
- [ ] 受邀参与农业政策咨询
- [ ] 培养农业技术人才

### 农业企业总经理 / 农业集团VP

- [ ] 管理年产值亿元级农业企业
- [ ] 农业产业链整合（种植→加工→销售→品牌）
- [ ] 融资和资本运作（农业PE/上市农企）
- [ ] 国际农业合作（海外农业投资）

## 👑 第5级 · 农业精英

### 精英标志
- 🏆 **袁隆平级别**：农业科技重大突破，改变国家粮食安全
- 🏆 **农业独角兽创始人**：打造估值10亿美元+的AgTech公司
- 🏆 **国家农业政策制定参与者**：参与制定粮食安全战略
- 🏆 **国际粮农组织（FAO）专家**：全球农业发展影响力
- 🏆 **农产品国际品牌创始人**：在国际市场建立知名农业品牌
